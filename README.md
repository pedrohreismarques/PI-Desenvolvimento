# Projeto Integrador - Desenvolvimento

Para esse projeto, recebemos o desafio de desenvolver uma aplicação Web onde ela faz o cadastro das pessoas que assistiram a apresentação dos projetos (desde os funcionários e alunos até os visitantes). Para a execução deste projeto, utilizamos as seguintes tecnologias e linguagens:

## Front-end
- HTML5;
- CSS3;
- JavaScript;
- Bootstrap.

## Back-end
- SQL (MySQL);
- Java;
- JavaScript.
