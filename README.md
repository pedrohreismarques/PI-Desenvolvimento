# Projeto Integrador - Desenvolvimento

Para esse projeto, recebemos o desafio de desenvolver uma aplicação Web onde ela faz o cadastro das pessoas que assistiram a apresentação dos projetos (desde os funcionários e alunos até os visitantes) e uma aplicação que realizará sorteios utilizando o número de identificação fornecido após o cadastro. Para a execução deste projeto, utilizamos as seguintes tecnologias e linguagens:

## Front-end
- HTML5;
- CSS3;
- JavaScript;
- Bootstrap.

### Cores
- Azul vívido (#03388f);
- Azure escure (#053969);
- Branco (#ffffff);
- Laranja (#df6f20);
- Preto (#000000).

## Back-end e Banco de Dados
- SQL (MySQL);
- Java;
- JavaScript.
